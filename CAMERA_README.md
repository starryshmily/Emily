# ESP32-C3 Camera功能说明

## 功能概述

ESP32-C3作为K230的显示终端，通过WiFi连接到K230，实时显示视频流和3D扫描进度。

## 配置步骤

### 1. 修改K230 IP地址

编辑 `ui_camera.c` 第182行，将K230的IP地址改为实际值：

```c
k230_client_config_t k230_config = {
    .host = "192.168.1.100",  // 修改为K230的实际IP地址
    .http_port = 8080,
    .ws_port = 8081,
    .is_connected = false
};
```

### 2. 获取K230的IP地址

在K230的CanMV IDE终端中查看启动日志，找到类似：
```
[WiFi] 连接成功! IP: 192.168.1.100
```

## 使用方法

1. **确保K230和C3在同一WiFi网络**
2. **启动K230** - 在CanMV IDE中运行main.py
3. **修改IP地址** - 在ui_camera.c中配置K230的IP
4. **编译C3固件** - 使用idf.py build
5. **烧录运行** - 使用idf.py flash
6. **点击Camera按钮** - 进入Camera页面
7. **点击"开始扫描"** - 开始3D扫描流程

## 网络通信协议

### WebSocket消息格式

**C3 → K230:**
```json
{"type": "start_scan"}
```

**K230 → C3 (进度更新):**
```json
{
    "type": "progress",
    "progress": 45,
    "message": "上传照片 (3/8)...",
    "stage": "uploading"
}
```

## 故障排查

### 无法连接到K230

- 检查K230是否已启动并连接WiFi
- 确认IP地址是否正确
- 确认两个设备在同一WiFi网络

### 视频流无法显示

- 检查HTTP端口8080是否可访问
- 浏览器测试: `http://K230_IP:8080/stream`

### 进度不更新

- 检查WebSocket连接是否正常
- 查看C3串口日志
