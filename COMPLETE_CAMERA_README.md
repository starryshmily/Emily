## 配置说明

### 1. K230配置

编辑 `D:\Emily\K230\config.py`：

```python
# WiFi配置
WIFI_SSID = "你的WiFi名称"
WIFI_PASSWORD = "你的WiFi密码"

# 混元3D API配置
HUNYUAN3D_API_KEY = "你的API密钥"
```

### 2. ESP32-C3配置

C3会自动通过mDNS发现K230，无需手动配置IP。

如果需要手动指定IP，编辑 `ui_camera.c` 第185行：

```c
.host = "192.168.1.100",  // 改为K230的实际IP
```

## 使用步骤

### 1. 启动K230

1. 打开CanMV IDE K230
2. 打开 `D:\Emily\K230\main.py`
3. 连接K230开发板
4. 点击 `下载并运行`

### 2. 启动ESP32-C3

1. 编译固件：`idf.py build`
2. 烧录固件：`idf.py flash`
3. 查看日志：`idf.py monitor`

### 3. 使用流程

1. **连接WiFi**：C3自动连接已保存的WiFi
2. **点击Camera**：进入Camera页面
3. **自动发现K230**：mDNS自动发现K230设备
4. **显示视频流**：实时显示K230摄像头画面
5. **点击开始扫描**：触发3D扫描流程
6. **查看进度**：实时显示扫描进度（0-100%）

## 网络协议

### WebSocket消息格式

**C3 → K230 (开始扫描):**

## 故障排查

### K230未发现

- 确认K230已启动并连接WiFi
- 查看K230终端日志获取IP地址
- 确认两个设备在同一WiFi网络

### 视频流无法显示

- 浏览器测试: `http://K230_IP:8080/stream`
- 检查ESP32-C3内存是否足够
- 查看串口日志中的错误信息

### 进度不更新

- 检查WebSocket连接状态
- 确认K230的WebSocket端口(8081)可访问
